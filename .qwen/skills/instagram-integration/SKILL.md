# Instagram Integration Skill

Monitor and post to Instagram automatically.

## Quick Reference

### Login (First Time)

```bash
cd watchers
python instagram_watcher.py --login
```

### Run Watcher

```bash
python instagram_watcher.py ../AI_Employee_Vault
```

### Post to Instagram

```bash
python instagram_watcher.py ../AI_Employee_Vault --post "Your caption"
```

## Features

- ✅ Monitor notifications
- ✅ Monitor messages
- ✅ Post with images
- ✅ Create action files for activity
